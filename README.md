# credit-risk-scorer
End-to-end credit default prediction model using LightGBM, SMOTE, and SHAP explainability — deployed as a live Streamlit web app
A machine learning project that predicts the probability of a borrower 
defaulting on a loan using the Give Me Some Credit dataset (150,000 records). 
Built with LightGBM, handles class imbalance via SMOTE, and includes SHAP-based 
explainability and a fairness audit across age groups. Deployed as an interactive 
Streamlit app where users can input borrower details and receive an instant risk 
score with a feature-level explanation.
